# 🚀 Deploy to Vercel (2 Minutes)

✅ **Code is already on GitHub!**

Your portfolio is at: https://github.com/Gunjanmehta001/Potfolio

Now deploy to Vercel with these steps:

---

## Step 1: Go to Vercel Dashboard

Visit: https://vercel.com/new

---

## Step 2: Import Your Repository

1. Click **"Import Git Repository"**
2. In the URL field, paste:
   ```
   https://github.com/Gunjanmehta001/Potfolio
   ```
3. Click **"Continue"**

---

## Step 3: Configure Project

- **Project Name**: `portfolio` (or keep default)
- **Framework**: Next.js (auto-detected ✅)
- **Root Directory**: `./` (default)
- Click **"Deploy"**

---

## Step 4: Wait for Deployment

Vercel will:
1. Clone your repo
2. Install dependencies
3. Build the Next.js app
4. Deploy to CDN

⏱️ Takes about **60 seconds**

---

## Step 5: Your Site is Live! 🎉

Vercel will show you a URL like:
```
https://potfolio-xxxxx.vercel.app
```

**This is your live portfolio URL!**
